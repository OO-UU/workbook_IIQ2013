En primer lugar se deben cargar las librerias que vamos a utilizar

Pregunta 2

Si la altura del estanque cilíndrico es de \(2\hspace{1mm}(m)\), la razón entre el diámetro del agitador y el diámetro del estanque es \(S1 = D/T = 0.2\), y el número del flujo del agitador es \(NQ = 0.87\), ¿Cuál es el caudal volumétrico descargado del borde del agitador operando a \(120\hspace{1mm}RPM\)?

  1. Q \(<\) 0.15 m\(^3\)/s

  2. 0.15 m\(^3\)/s \(\leq\) Q \(<\) 0.20 m\(^3\)/s

  3. 0.12 m\(^3\)/s \(\leq\) Q \(<\) 0.25 m\(^3\)/s

  4. Q \(\geq\) 0.25 m\(^3\)/s

%%render long
H = 2 #m
V = 3 #m$^3$
S_1 = 0.2
D_t = sqrt(4*V/(pi*H)) #m
D_a = D_t*S_1 #m
NQ = 0.87
N = 120 #RPM
Q = NQ*(N/60)*D_a**3 #m$^3$/s

\[ \begin{aligned} H &= 2 \; \;\textrm{(m)} \\[10pt] V &= 3 \; \;\textrm{(m$^3$)} \\[10pt] S_{1} &= 0.200 \; \\[10pt] D_{t} &= \sqrt { 4 \cdot \frac{ V }{ \pi \cdot H } } \\&= \sqrt { 4 \cdot \frac{ 3 }{ 3.142 \cdot 2 } } \\&= 1.382 \; \;\textrm{(m)}\\[10pt] \\[10pt] D_{a} &= D_{t} \cdot S_{1} \\&= 1.382 \cdot 0.200 \\&= 0.276 \; \;\textrm{(m)}\\[10pt] \\[10pt] \mathrm{NQ} &= 0.870 \; \\[10pt] N &= 120 \; \;\textrm{(RPM)} \\[10pt] Q &= \mathrm{NQ} \cdot \left( \frac{ N }{ 60 } \right) \cdot \left( D_{a} \right) ^{ 3 } \\&= 0.870 \cdot \left( \frac{ 120 }{ 60 } \right) \cdot \left( 0.276 \right) ^{ 3 } \\&= 0.037 \; \;\textrm{(m$^3$/s)}\\[10pt] \end{aligned} \]

La alternativa correcta es la a.

Pregunta 3

El fabricante del agitador reporta que para un diámetro del agitador de \(0.3\hspace{1mm}m\), la potencia requerida para \(180\hspace{1mm}RPM\) es de \(5.5 \hspace{1mm}kW\). ¿Qué potencia requiere el agitador diseñado en la pregunta anterior?

  1. P \(<\) 0.15 kW

  2. 0.5 kW \(\leq\) P \(<\) 1.5 kW

  3. 1.5 kW \(\leq\) P \(<\) 2.5 kW

  4. P \(\geq\) 2.5 kW

Se debe cumplir que el número de potencia es equivalente.

\[ N_{p,1} = N_{p,2} \Rightarrow \frac{P_1}{\rho_1N_1^3D_{a,1}^5} = \frac{P_2}{\rho_2N_2^3D_{a,2}^5} \]

%%render long
N_1 = 180 #RPM
N_2 = N #RPM
P_1 = 5.5 #kW
D_a1 = 0.3 #m
D_a2 = D_a #m
P_2 = P_1*((N_2/60)**3*D_a2**5)/((N_1/60)**3*D_a1**5) #kW

\[ \begin{aligned} N_{1} &= 180 \; \;\textrm{(RPM)} \\[10pt] N_{2} &= 120 \; \;\textrm{(RPM)} \\[10pt] P_{1} &= 5.500 \; \;\textrm{(kW)} \\[10pt] D_{a1} &= 0.300 \; \;\textrm{(m)} \\[10pt] D_{a2} &= 0.276 \; \;\textrm{(m)} \\[10pt] P_{2} &= P_{1} \cdot \frac{ \left( \frac{ N_{2} }{ 60 } \right) ^{ 3 } \cdot \left( D_{a2} \right) ^{ 5 } }{ \left( \frac{ N_{1} }{ 60 } \right) ^{ 3 } \cdot \left( D_{a1} \right) ^{ 5 } } \\&= 5.500 \cdot \frac{ \left( \frac{ 120 }{ 60 } \right) ^{ 3 } \cdot \left( 0.276 \right) ^{ 5 } }{ \left( \frac{ 180 }{ 60 } \right) ^{ 3 } \cdot \left( 0.300 \right) ^{ 5 } } \\&= 1.082 \; \;\textrm{(kW)}\\[10pt] \end{aligned} \]

La alternativa correcta es la b.
